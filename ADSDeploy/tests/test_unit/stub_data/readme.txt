The folder holds stub data for the unittests. Make sure you record how the data was generated.

The folder is referenced by the unittests (eg: app.config.get("TEST_UNIT_DIR"))

stub_webapp
-----------

Obtained from GitHub, an example webhook JSON body that they send. The fastest
way to obtain one is to go to GitHub, the adsabs repository, click the 'cog' in
the corner to go to the settings dashboard. You should then see a webhooks button,
which directs you to the webhook settings. You can then click an example webhook
that has already been sent, and obtain the entire JSON body.
